# CodeAlpha Full-Stack Web Development Projects

This repository contains my official task submissions for the CodeAlpha Full-Stack Web Development Internship track. All web servers and backend database routes were engineered and deployed entirely within a local mobile sandbox environment using Python and Flask.

## ⚙️ Project Architectures

### # Task 1: Cosmetics E-Commerce Storefront
- Built a responsive web application for an independent makeup brand.
- Implemented server-side array tracking to manage active shopping cart states across dynamic pages.

### # Task 2: Micro-Blogging Social Media Feed
- Engineered an interactive community timeline layout using Flask HTML rendering.
- Developed real-time input fields allowing users to publish and display timeline posts instantly.

### # Task 3: Agile Project Management Workspace
- Constructed a collaborative team task planner tracking statuses, descriptions, and assignees.
- Configured dynamic comment routing tables allowing team members to append notes to specific project cards.
